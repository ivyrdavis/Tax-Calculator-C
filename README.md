# Tax-Calculator-C
Federal income tax calculator written in C.

Overview

This project is a C program that calculates U.S. federal income tax liability based on the federal tax code. The application supports multiple filing statuses and calculates taxable income using user-provided financial information.

This project was completed as part of a computer science course at the University of Alabama.

Features
- Supports multiple filing statuses
- Calculates taxable income
- Applies progressive federal tax brackets
- Processes deductions
- Console-based user interface

Skills Demonstrated
- C Programming
- Functions
- Conditional Statements
- User Input Validation
- Algorithm Design
- Problem Solving

Technologies
- C
- GCC
- Visual Studio Code

Learning Outcomes
Through this project I gained experience implementing real-world financial rules in software and translating tax regulations into structured program logic.
